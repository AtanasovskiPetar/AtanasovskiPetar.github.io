Info

Building a Real Time Sign Language Detection model with Tensorflow Object Detection API, Python and Deep Learning.

The Challenge

Develop a real-time sign language recognition model utilizing your device's webcam.

The Solution

We utilized the Tensorflow Object Detection API along with a pre-trained model named SSD MobNet V2. We gathered images of us showing sign language from, labeled them, and proceeded to train the model. After optimization, the model successfully recognized sign language in real-time frames captured from the webcam.
